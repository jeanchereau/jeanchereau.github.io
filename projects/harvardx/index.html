Interactive visualization of the binomial distribution by Jean Chereau (London, UK). More info...

Descripton

The binomial distribution is the discrete probability distribution of the number of successes in a sequence of n independent yes/no experiments (draws), each of which yields success with probability p (success probability).

Imagine a biased coin which comes up heads with probability 0.3 (success probability) when tossed. Toss it as many times as you want (draws).

Click Run to run the experiment.

Parameters

Probability mass function
Cumulative distribution function
